$typekit-font

scss
$typekit-font: (
  'name': 'freight-text-pro',
  'stack': 'fantasy',
  'source': 'https://typekit.com/fonts/freight-text',
);

Description

A font imported from typekit.

  • call the specimen preview directly
  • give a font-name and parenthetical list of variants on the first line
  • suppy additional html head as a nested block

freight-text-pro (typekit)

Stack: fantasy

$google-font

scss
$google-font: (
  'name': 'Exo 2',
  'source': 'https://fonts.google.com/specimen/Exo+2',
  'stack': ('arial', 'monospace', 'fantasy'),
);

Description

A font imported from google fonts.

  • call the specimen preview directly
  • give a font-name and parenthetical list of variants on the first line
  • suppy additional html head as a nested block

Exo 2 (google)

Stack: arial, monospace, fantasy

$local-font

scss
$local-font: (
  'name': 'rockingham',
  'regular': 'rockingham/rockingham-regular-webfont',
  'bold': 'rockingham/rockingham-bold-webfont',
  'italic': 'rockingham/rockingham-italic-webfont',
  'bold' 'italic': 'rockingham/rockingham-bolditalic-webfont',
  'stack': 'fantasy',
);

Description

A locally-hosted font.

  • call the specimen preview directly
  • give a font-name and bracketed list of formats on the first line
  • optionally override variants with parenthetical list
  • no additional html head
  • add a font-path global config option, similar to sass path options

rockingham (local)

Stack: fantasy
